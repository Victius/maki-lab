1.1 Definition of Groups

definition of groups

​ Now we start our trips in the world of abstract algebra (the basic part). ​ The first chapter, groups, tell us we will contact with a new thing called the group. Let's begin with the set, which everyone is familiar with. ​ A simple set, which does not have the distinct definition, may contains something ('elements'), and need not hold other properties, for example, a class is a set that contains many students. As everyone knows, however, many sets also contains some 'relations' and 'operations', such as $\R$, which contains addition, multiplication, 'little than', and so on. We will focus on the '(binary) operation'.

Definition 1.1 binary operation/ rule of combination ​ A binary operation is a map: $f:A\times B\to C,\ (a,b)\mapsto f(a,b)$. ​ In particular, if we let $B, C$ equal $A$: $f:A\times A\to A$, we called the binary operation on set $A$.

we have learned many binary operations before:

Example 1 addition, subtraction, multiplication on $\R$. (where is the 'division'?)

Example 2 addition, subtraction on vector space $\R^{n}$.

Example 3 for vectors in $R^{n}$, the 'dot product': $\cdot:\R^n\times \R^n\to \R$

​ And now, we will give the definition of groups:

Definition 1.2 groups, $(G,\cdot)$ ​ A group is a set $G$ with a binary operation ''$\cdot$'' on $G$, satisfing the following axioms: ​ (1) (the associative law) for all $a,b,c\in G$, $(a\cdot b)\cdot c=a\cdot(b\cdot c)$. ​ (2) (identity) there is an element $e\in G$ s.t. ('such that') for all $a\in G$, $ae=ea=a$. $e$ is called an identity of $G$. ​ (3) (inverse) for all $a\in G$, there is an element $b\in G$ s.t. $ab=ba=e$. $b$ is called an inverse of $a$.

​ Without confusion, we will write $G$ instead of $(G,\cdot)$ to express a group, and use $ab$ instead of $a\cdot b$.

​ The definition of groups seem clearly, but you may not know what it is. This definition is a typical 'abstract' definition, which means it does not tell you a specific object, but some properties to restrict something, and get a kind of new objects. We will give some examples for 'groups', and you will find the advantage of the abstract definition.

Example 4 the integer set $\Z$, with addition $+$:

| $\forall i,j\in\Z, i+j\in\Z$ | | ----------------------------------------------- | | $\forall i,j,k\in\Z, (i+j)+k=i+(j+k)$ | | $\exist 0\in\Z,\forall n\in\Z,n+0=0+n=n$ | | $\forall n\in\Z,\exist -n\in\Z,n+(-n)=(-n)+n=0$ |

​ Thus $(\Z,+)$ is a group.

Ex 1.1.1 how about $(\N,+),(\Q,+),(\R,+),(\C,+)$ ?

Example 5 the positive rational number set $\Q_+$, with multiplication $\cdot$:

| $\forall q,r\in\Q+, qr\in\Q+$ | | ------------------------------------------------------- | | $\forall q,r,s\in\Z, (qr)s=q(rs)$ | | $\exist 1\in\Q+,\forall q\in\Z,q\cdot1=1\cdot q=q$ | | $\forall q\in\Q+,\exist q^{-1}\in\Z,qq^{-1}=q^{-1}q=1$ |

​ Thus $(\Q_+,\cdot)$ is a group.

Ex 1.1.2 how about $(\Z_+,\cdot),(\Q,\cdot)$?

Example 6 the set of all $m\times n$ real matrix $\R^{m\times n}$, with addition $+$ is a group.

Example 7 the set of all function $f:\R\to \R$, $\R^{\R}$, with addition '$+$' defined $$ \forall x\in \R, (f+g)(x)=f(x)+g(x) $$ ​ Then $(\R^{\R},+)$ is a group.

​ Maybe you have already found the advantage of the abstract definition: it can describe a lot of objects which have the same properties ('axioms'). If we want to know some other properties in these objects, just research the abstract one instead of them all! ​ Some reader may find that, in example 1.5-1.7, an another property exist, the commutative law: for all $a,b\in G$, $a+b=b+a$, why does it not appear in the three axioms of groups? Because we have some other examples:

Example 8 general linear group of degree $n$, $GLn(F),\ F=\Q,\R$ or $\C$. $$ GLn(F) = {A\in F^{n\times n}|\ \det A\neq 0} $$ ​ It contains all $n\times n$ matrix, which the determinant of $A$ is not zero. We consider the 'multiplication' of the two matrices $A,B\in GL_n(F)$:

| $\forall A,B\in GLn(F), AB\in GLn(F)$ | | ------------------------------------------------------------ | | $\forall A,B,C\in GLn(F), (AB)C=A(BC)$ | | $\exist In\in GLn(F),\forall A\in GLn(F),AIn=InA=A$ | | $\forall A\in GLn(F),\exist A^{-1}\in GLn(F),AA^{-1}=A^{-1}A=I_n$ |

​ Thus $(GLn(F),\cdot)$ is a group. ​ Consider the commutative law. Obviously, for arbitrary $A,B\in GLn(F)$, $AB\neq BA$ ! The commutative law is not necessary.

Definition 1.3 commutative groups/ Abelian groups ​ If a group also satisfies the commutative laws, we say it is a commutative group, or an Abelian groups.

​ Groups in example 3 to 7 are all Abelian groups.

Ex 1.1.3 define special linear group of degree $n$, $SLn(F), F=\Q,\R$ or $\C$. $$ SLn(F) = {A\in F^{n\times n}|\ \det A=1} $$ ​ How about $(SL_n(F),\cdot)$? does it satisfies the commutative law?

​ Now we will deduce some properties of groups, from the three axioms, in the next section, we will see more and important examples.

Properties of groups

Property 1.4 In group $G$, there is a unique identity. ​ Generally speaking, if you want to say something is unique, you can firstly assume two things like it, and try to prove that they are the same. ​ Proof: let $e,e'$ are identities of $G$, then we have: $$ \begin{align} e &= ee'&\text{($e'$ is an identity)}\ &= e' &\text{($e$ is an identity)} \end{align} $$

Property 1.5 In group G, for arbitrary $a\in G$, there is a unique inverse element of $a$. ​ Proof: let $b,b'\in G$ are inverse elements of $a$, then we have: $$ \begin{align} b &= be &\text{(inentity)}\ &= b(ab') &\text{(inverse)}\ &= (ba)b' &\text{(the associative law)}\ &= eb' &\text{(inverse)}\ &= b' &\text{(inentity)}\ \end{align} $$ ​ Because of this uniqueness, we give a symbol, $a^{-1}$, to denote the inverse of $a$.

Ex 1.1.4 In group $G$, $a\in G$, then $(a^{-1})^{-1}=a$. (Property 1.6)

Ex 1.1.5 [*, !] Let the set $A$ contains a binary operation '$\cdot$', which satisfies the associative law, and $A$ has an identity $e$, then $b\in A$ is the inverse of $a$ iff. $aba=a$ and $ab^2a=e$.

Property 1.7 In group $G$, for all $a,b\in G$, we have $(ab)^{-1}=b^{-1}a^{-1}$. ​ Proof: calculate $(ab)(b^{-1}a^{-1})$ and $(b^{-1}a^{-1})(ab)$, then they all equal to $e$. Thus $(b^{-1}a^{-1})$ is the inverse of $ab$. Also, $(ab)^{-1}$ is obviously the inverse of $ab$. In conclusion, $(ab)^{-1}=b^{-1}a^{-1}$.

Property 1.8 In group $G$, for all $a,b\in G$, the equation $ax=b$ have solutions in $G$. $ya=b$ is the same. ​ Proof: it is easy to find that $x=a^{-1}b$ is a solution of $ax=b$.

Ex 1.1.6 In group $G$, if $a,x, y\in G$ s.t. $ax=ay$, then $x=y$. $xa=ya$ is the same. (Property 1.9: the cancellation law )

Other Definition of Groups

​ In history, mathematicians always left definitions and theorems become shorter and shorter, for example the 'postulate of parallel lines' in geometry. Groups also have some briefer definitions. We introduce a useful one.

Definition 1.10 Groups, 'the left side' A group is a set $G$ with a binary operation ''$\cdot$'' on $G$, satisfying the following axioms: (1) (the associative law) for all $a,b,c\in G$, $(a\cdot b)\cdot c=a\cdot(b\cdot c)$. (2) (left identity) there is an element $e\in G$ s.t. ('such that') for all $a\in G$, $ea=a$. (3) (left inverse) for all $a\in G$, there is an element $a^{-1}\in G$ s.t. $a^{-1}a=e$.

​ We need to prove this definition is equivalent to the definition 1.2, that is to say, we need to prove $ae=a$ and $aa^{-1}=e$ from definition 1.10. ​ Proof: firstly $$ \begin{align} aa^{-1} &= e(aa^{-1}) &\text{(left identity)}\ &= ((a^{-1})^{-1}a^{-1})(aa^{-1}) &\text{(left inverse)}\ &= (a^{-1})^{-1}(a^{-1}a)a^{-1} &\text{(the associative law)}\ &= (a^{-1})^{-1}ea^{-1} &\text{(left inverse)}\ &= (a^{-1})^{-1}(ea^{-1}) &\text{(the associative law)}\ &= (a^{-1})^{-1}a^{-1} &\text{(left identity)}\ &= e &\text{(left inverse)}\ \end{align} $$ ​ secondly $$ \begin{align} ae &= a(a^{-1}a) &\text{(left inverse)}\ &= (aa^{-1})a &\text{(the associative law)}\ &= ea &\text{(prove above!)}\ &= a &\text{(left identity)}\ \end{align} $$
