1.6 Homomorphisms

The definition of homomorphisms

​ In section 1.3 to 1.5, we deeply discuss the subgroup, which are all in just one group. Today we will begin with two groups together, which means a kind of specific map between two groups. Let's have a look at a canonical example first.

Example 1 determinant

​ In linear algebra, we have learned for all $A,B \in F^{n\times n}$, $$ \det A\cdot\det B = \det AB $$ Now we restrict the discussion limit into $GLn(\C)$. In fact, we can see the 'determinant' as a map: $$ \begin{align} \det:(GLn(C),\cdot) &\to (\C^*,\cdot)\ A &\mapsto \det(A) \end{align} $$ Thus it is a map between two groups. (Notice that two operations are not the same.) Also, it has a 'parallel relation'

| $(GL_n(C),\cdot)$ | $(C^*,\cdot)$ | | ----------------- | ------------------------------ | | $A$ | $\det A$ | | $B$ | $\det B$ | | $AB$ | $\det AB = \det A\cdot \det B$ |

The equation $\det AB = \det A\cdot \det B$ is not trivial, we need to give it a definition.

Definition 6.1 homomorphisms ​ Let $(G,\cdot)$ and $(H,\circ)$ are two groups. A map $f:G\to H$ s.t. for all $x,y\in G$ $$ f(a\cdot b) = f(a)\circ f(b) $$ is called a homomorphism.

​ We will see more examples of homomorphisms.

Example 2 If $H$ is a subgroup of $G$, then we just define a embedding map $$ \begin{align} i:H &\to G \ h &\mapsto h \end{align} $$ then it is a homomorphism because $$ \begin{align} i(g)+i(h) &= g+h \ &= i(g+h) \end{align} $$ ​ This is a simple but important homomorphism.

Example 3 Define the map $$ \begin{align} f:(\Z,+) &\to (\Z/n\Z,+) \ i &\mapsto \bar i \end{align} $$ then it is a homomorphism because $$ \begin{align} f(i)+f(j) &= \bar i+\bar j \ &= (i+n\Z)+(j+n\Z) \ &= (i+j)+n\Z &(n\Z+n\Z=n\Z) \ &= \overline{i+j} \ &= f(i+j) \end{align} $$

Example 4 We have known that if define $U = {x\in \C|\ x^n=1}$, then $(U,\cdot)$ will be a group.(Section 1.2) Let $\sigma=e^{2\pi/n}$, and define a map: $$ \begin{align} f: (\Z/n\Z) &\to (U,\cdot) \ \bar i &\mapsto \sigma^i \end{align} $$ then it is a homomorphism because $$ \begin{align} f(\bar i)+f(\bar j) &= \sigma^i\cdot\sigma^j \ &= \sigma^{i+j} \ &= f(\overline{i+j}) \ &= f(\bar i+\bar j) \end{align} $$ ​ Furthermore, we can find that $f$ is a bijection, then there is obviously an inverse map $$ \begin{align} f^{-1}: (U,\cdot) &\to (\Z/n\Z) \ \sigma^i &\mapsto \bar i \end{align} $$ and it is also a homomorphism! [Ex 1.6.1]

​ In example 4 we can learn that a homomorphism with a bijection will have more beautiful properties, thus we will give it a name.

Definition 6.2 isomorphisms/ isomorphic, $G\cong H$ Let $f:G\to H$ is a (group) homomorphism. If $f$ is a bijection, then we will call $f$ is an isomorphism. And $G$ and $H$ are said to be isomorphic, written $G\cong H$.

Example 5 The linear map ​ For two linear space $V,W$ and a linear map $T:V\to W$, if we view linear spaces as groups and only consider the addition, then $T$ is a homomorphism. What's more, if $\dim V=\dim W$, then $T$ is an isomorphism. [Ex 1.6.2]

Example 6 projections of the direct product

​ Let $G1,\cdots, Gn$ are groups. Then define the $i^{th}$ projection of the direct product $G1\times\cdots\times Gn$ $$ \begin{align} \pii:G1\times\cdots\times Gn &\to Gi \ (g1,\cdots,gn) &\mapsto g_i \end{align} $$ Then this map is a homomorphism. [Ex 1.6.3]

Some basic properties of homomorphisms

Theorem 6.3 keep the identity and inverse ​ Let $f:G\to H$ is a homomorphism. $e$ is the identity of $G$, and $e'$ is the identity of $H$. We have ​ (1) $f(e)=e'$. ​ (2) If $g\in G$, then $f(g^{-1})=(f(g))^{-1}$.

​ Proof: (1). $f(e) = f(e\cdot e)=f(e)f(e)$, thus $e'=f(e)$. ​ (2). $$ \begin{align} f(g^{-1}) &= f(g^{-1})e' \ &= f(g^{-1})\cdot f(g)(f(g))^{-1} \ &= f(g^{-1})f(g)\cdot (f(g))^{-1} \ &= f(g^{-1}g)\cdot (f(g))^{-1} \ &= f(e)\cdot (f(g))^{-1} \ &= e'\cdot (f(g))^{-1} \ &= (f(g))^{-1} \end{align} $$

Ex 1.6.6 Let $f:G\to H$ is a homomorphism. If $g$ is in $G$, then for arbitrary $n\in\Z$, $f(g^n)=(f(g))^n$.

Review: null and image space of a linear map

​ Before our further learning, we'd like to review some important things about linear maps in linear algebra. There are two subspaces with respect to a linear map $T: V\to W$, null space (or kernel) and image space (or range): $$ \ker T = {\vec v\in V|\ T\vec v=\vec 0} \ \mathrm{im} T = {T\vec v\in W|\ \forall\vec v\in V } $$ And they have some properties (due to space limitations, detailed proof is not given here, you can review in any textbook of linear algebra.)

Theorem L.1 subspaces ​ (1) null space $\ker T$ is a subspace of domain $V$. ​ (2) image space $\mathrm{im} T$ is a subspace of co-domain $W$.

Theorem L.2 injective and surjective ​ (1) A linear map $T: V\to W$ is injective, iff. $\ker T={\vec 0}$. ​ (2) A linear map $T: V\to W$ is surjective, iff. $\mathbb{im} T=W$.

Theorem L.3 nullity+rank (1) For arbitrary linear map $T: V\to W$, $\dim V = \dim\ker T + \dim\mathbb{im}T$. (2) Thus, for respective quotient space $T/\ker T$, $\dim(T/\ker T) = \dim \mathbb{im}T$, then $T/\ker T \cong \mathbb{im}T$.

​ When we only consider the addition in a linear space, we can deduce it is an Abelian group, and a linear map is a homomorphism in respective group. So can we find parallel definitions and theorems, which are mentioned above, in group theory? Try to do!

Kernel and image

​ Just like null and image space, we will define the 'similar' things with respect to a (group) homomorphism (Notice that in this section, we will omit the word 'group' in order to keep brevity).

Definition 6.4 kernel and image ​ For a homomorphism $f:G\to H$, define the kernel of $f$ which includes all elements in $G$ that map to the identity $e'$ of $H$. $$ \ker f = {g\in G|\ f(g) = e'} $$ ​ We also define the image of $f$ which includes all images under $f$ of elements in $G$. $$ \mathrm{im} f = {f(g)|\ \forall g\in G} $$

​ Now we come back to examples 1 to 4, and try to find the kernel and image of those homomorphisms.

Example 1' For the homomorphism $\det:(GLn(C),\cdot) \to (\C^*,\cdot)$: $$ N := \ker \det = {A\in GLn(C)|\ \det(A)=1} $$ then the kernel $N$ is $SL_n(C)$. And the image $I$ is the whole $\C^\ast$, because for all $z\in\C^\ast$, $\det(\mathrm{diag}(z,1,\cdots,1)) = z$.

Example 2' For the embedding homomorphism $i:H \to G$, $N = {e}, I = H$. [Ex 1.6.7]

Example 3' For the homomorphism $f:(\Z,+) \to (\Z/n\Z,+)$: $$ \begin{align} N &= {i\in\Z|\ \bar i = \bar 0} \ &= {i\in\Z|\ i \in n\Z} \ &= n\Z \ \end{align} $$ And the image $I$ is the whole $\Z/n\Z$.

Example 4' Similar discussions leave to reader. [Ex 1.6.8]

Example 6' For the $i^{th}$ projection $\pii$, $N=G1\times\cdots\times{ei}\times\cdots\times Gn \cong G1\times\cdots\times G{i-1}\times G{i+1}\times\cdots\times Gn$ and $I=G_i$. [Ex 1.6.9]

Theorem 6.5 subgroups with respect to a homomorphism ​ Let $f:G\to H$ is a homomorphism, then we have ​ (1) $\ker f \lhd G$. ​ (2) $\mathrm{im} f \leqslant H$.

​ Proof: (1) (i) wts. $\ker f \leqslant G$. ​ Obviously, the identity $e$ of $G$ is in $\ker f$. (6.3 (1), why we need to say this point in proof?) ​ For all $a,b\in \ker f$, we have: $$ \begin{align} f(ab^{-1}) &= f(a)f(b^{-1}) &\text{(6.1)}\ &= f(a)(f(b))^{-1} &\text{(6.3 (2))}\ &= e'(e')^{-1} \ &= e' \end{align} $$ thus, $\forall a,b \in \ker f,\ ab^{-1}\in \ker f$, i.e. $\ker f\leqslant G$. (3.3) ​ (ii) wts. for all $g\in G$, $g(\ker f)g^{-1}\subset \ker f$. (then for 5.3, $\ker f\lhd G$) ​ For all $a\in \ker f$, we have $$ \begin{align} f(gag^{-1}) &= f(g)f(a)f(g^{-1}) &\text{(6.1)} \ &= f(g)e'(f(g))^{-1} &\text{(6.3 (2))} \ &= f(g)(f(g))^{-1} \ &= e' \end{align} $$ so $\forall a\in \ker f, gag^{-1}\in \ker f$. That is to say, $g(\ker f)g^{-1}\subset \ker f$.

​ (2) Obviously, $e'$ is in $\mathrm{im} f$. (6.3 (1)) ​ For all $x,y\in \mathrm{im}f$ (i.e. there are some $a,b\in G$ s.t. $f(a) = x, f(b) = y$), we have $$ \begin{align} xy^{-1} &= f(a)(f(b))^{-1} \ &= f(a)f(b^{-1}) &\text{(6.3 (2))} \ &= f(ab^{-1}) &\text{(6.1)} \end{align} $$ so $\forall x,y \in \mathrm{im}f, xy^{-1}\in\mathrm{im}f$, i.e. $\mathrm{im}f\leqslant H$. (3.3)

Theorem 6.6 injective and surjective ​ Let $f:G\to H$ is a homomorphism. $e$ is the identity of $G$, and $e'$ is the identity of $H$. Then we have: ​ (1) A homomorphism $f:G\to H$ is injective, iff. $\ker f={e}$. ​ (2) A homomorphism $f:G\to H$ is surjective, iff. $\mathbb{im} f=H$. (Actually it's the definition of the surjection. We put it here because it is parallel with (1).)

​ Proof: we only need to prove (1). ​ (i) '$\rightarrow$': If $f:G\to H$ is injective, then for all $a,b\in G$, $f(a)=f(b)$ leads to $a=b$. Thereby $f(ab^{-1})=e'$ leads to $ab^{-1}=e$, i.e. $ab^{-1}\in \ker f\to ab^{-1}=e$. So $\ker f\subset{e}$, and $\ker f = {e}$. ​ (ii) '$\leftarrow$': Let $\ker f={e}$. If $a,b\in G$, $f(a)=f(b)$, then $f(ab^{-1})=e'$. That is to say, $ab^{-1}=e$. So $a=b$. Thus $f$ is injective.

​ We can find the homomorphism in example 1, example 3 and example 6 are surjective, in example 2 is injective, and in example 4 is bijective (thus is an isomorphism).
